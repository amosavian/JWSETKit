# ``P256K``

The secp256k1 (P-256K) elliptic curve.

## Overview

`P256K` is the namespace enum for everything in this module. It exposes two
sub-namespaces — ``P256K/Signing`` for ECDSA and Schnorr, and
``P256K/KeyAgreement`` for ECDH.

See <doc:1-Overview> for an introduction to the curve and how this module is
organized.
